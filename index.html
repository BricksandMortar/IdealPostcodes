Accurate Address Verification for the UK

Address verification for Rock RMS using the most accurate data available.

Accurate

Ideal Postcodes uses data updated daily from the Royal Mail Postcode Address File as well as additional data from the ONS and Ordnance Survey.

Regularly Updated

This plugin takes advantages of all of the features of Ideal Postcodes Location Service and is regularly updated to be compatible with new features as they are released.

Open Source

Like all plugins produced by Bricks and Mortar, Ideal Postcodes Location Service is open source guaranteeing a transparent product you can rely on.
